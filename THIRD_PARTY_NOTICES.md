# Third-party notices

MouseBridge contains interoperability work informed by these open-source
projects:

- [Scroll Reverser](https://github.com/pilotmoon/Scroll-Reverser), Copyright
  Nicholas Moore, Apache License 2.0. Its public implementation documents the
  gesture timing approach for distinguishing trackpads from mice and the need
  to update both Quartz and underlying IOHID scroll values.
- [Solaar](https://github.com/pwr-Solaar/Solaar), GPL-2.0. Its public source was
  consulted as protocol documentation for Logitech HID++ adjustable-DPI list
  encoding. MouseBridge does not include or link Solaar code.

See each linked project for its complete license and notices.
