# Corresponding source / 对应源码

MouseBridge is distributed under GPL-3.0-or-later. The complete corresponding
source for each binary release, including build and installation scripts, is
available from the tag matching the application version:

https://github.com/gmch1/mousebridge-macos/releases

The GitHub Release page provides automatically generated source archives next
to the MouseBridge binary ZIP at no charge.

MouseBridge 使用 GPL-3.0-or-later 分发。每个二进制版本的完整对应源码（包括构建与安装
脚本）可从与应用版本一致的 tag 获取。GitHub Release 页面会在二进制 ZIP 旁免费提供
自动生成的源码归档。
